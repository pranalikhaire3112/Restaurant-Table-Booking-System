# Restaurant-Table-Booking-System
The Restaurant Table Booking System is a web-based application designed to simplify the table reservation process for restaurants and customers. It allows users to book tables online while enabling administrators to manage bookings, tables, and customers efficiently through an admin dashboard.This project helps reduce waiting time, avoids manual booking errors, and improves overall restaurant management.

# Features

# User Module - 

1.User table booking

2.Select date & time

3.View booking status

4.Simple and user-friendly interface

# Admin Module -

1.Secure admin login

2.Manage table bookings

3.Approve / reject reservations

4.Manage tables

5.View all booking details

6.Manage sub-admins

7.Password & profile update
