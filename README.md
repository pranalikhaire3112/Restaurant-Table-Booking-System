# Restaurant-Table-Booking-System
Developed a web-based Restaurant Table Booking System using PHP, MySQL, HTML to allow users to reserve tables online and check availability in real-time and admin management using a structured waterfall development model.

FEATURES
1. User Registration and Authentication
•	Sign-Up/Login: Users can create accounts using email, phone number, or social media credentials.
•	Profile Management: Customers can update personal details and preferences.
2. Restaurant Search and Filter
•	Search: Users can search for restaurants based on name, location, or cuisine.
•	Filters: Apply filters like rating, availability, distance, price range, and type of cuisine.
3. Real-Time Table Availability
•	Dynamic Availability: Display real-time table availability based on the restaurant's current status.
•	Capacity Management: Restaurants can set the number of tables and adjust them based on demand.
4. Booking Management
•	Instant Booking: Users can instantly book a table at their desired time slot.
•	Advanced Booking: Option for users to reserve tables for future dates.
•	Time Slot Selection: Select preferred time slots for dining.
•	Party Size: Option to select the number of guests for the booking.
5. Notifications and Reminders
•	Booking Confirmation: Users receive immediate confirmation of their bookings via email/SMS.
•	Reminders: Send reminders before the booking time.
•	Cancellation/Reschedule Alerts: Notifications for successful cancellations or changes in booking.
6. Special Requests
•	Customization: Users can add special requests like high chairs, birthday arrangements, or dietary preferences during the booking process.
7. Payment Integration 
•	Online Payment: Secure payment gateways for advance payment or reservation deposits.
•	Split Payment: Option to split payments among guests.
8. Restaurant Dashboard 
•	Table Management: Restaurants can manage available tables, peak times, and block off unavailable slots.
•	Customer Management: View customer bookings, cancellations, and preferences.
•	Analytics: Insights on peak hours, repeat customers, and feedback.
9. User Reviews and Ratings
•	Post-Dining Reviews: Customers can leave reviews and rate the restaurant based on their experience.
•	Restaurant Response: Restaurants can respond to reviews, fostering customer relationships.
10. Loyalty and Discounts
•	Loyalty Programs: Customers earn points on each booking or visit which can be redeemed for future discounts.
•	Promo Codes: Support for promotional offers and discount codes during the booking process.
11. Waitlist and Queuing System
•	Waitlist Management: If the restaurant is fully booked, users can join a waitlist and be notified when a table becomes available.
•	Queue Position: Real-time tracking of queue position if customers opt for walk-in.

12. Feedback and Customer Support
•	In-App Feedback: Customers can provide feedback or report issues related to their booking.
•	Customer Support Chat: An option for live customer support to assist with booking issues or inquiries.
13. Multi-Language and Localization
•	Language Support: Multi-language support for users from different regions.
•	Currency and Time Zone Adjustments: Based on user location.
14. QR Code for Check-In
•	Contactless Check-In: Users can scan a QR code upon arrival for contactless check-in, directly from the app.
